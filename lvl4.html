Hoja de Estilo en cascada (CSS)

CSS es el acronimo para Cascading Style Sheets que en español significa Hojas de Estilo en Cascada. A diferencia de HTML que define la estructura de una pagina web, CSS define la presentación, es decir como deben mostrarse los elementos en la pagina, su color, su forma, posición, espaciado, etc. En resumen, la estetica (estilo) del sitio web.

No podemos utilizar CSS sin HTML, ya que esta es una herramienta que modifica solo la estetica del sitio. Hoy en día, por la manera en que evolucionó la web, tampoco tiene mucho sentido utilizar HTML sin CSS. Esto lleva a dejar obsoletas algunas etiquetas.

Muchas etiquetas HTML se utilizan para dar formatos a textos, pero estas fueron reemplazadas con la utilización de CSS.

Partes principales de CSS

Hay muchos elementos con los que tenemos que trabajar para declarar los estilos, basicamente los más importantes son:

  • Selectores, mediante los cuales podemos especificar a que elementos de la página estamos haciendo referencia. Los principales selectores son: Selector de etiqueta (solo se escribe el nombre de la etiqueta), selector de clase (caracter .), selector id (#).
  • Atributos de estilo para definir que cosas queremos estilizar sobre los selectores indicados. Estos se indican dentro de las llaves.
  • Una serie de valores, que indican que estilo se debe aplicar a cada atributo sobre cada selector. Algunos valores se pueden expresar en unidades CSS, que sirven para cuantificar estos (pixeles, puntos, pulgadas, etc.). Estos se escriben a la derecha del atributo, luego del símbolo :

Combinadores

A travez de los selectores, podemos modificar las propiedades de diferentes elementos HTML según determinados criterios.

Nombre Simbolo Ejemplo Resultado
Combinador hijo > div > p Selecciona TODO elemento <p> que es hijo directo de un elemento <div>
Combinador de descendientes (espacio) div p Selecciona TODOS los elementos <p> dentro de un elemento <div>
Combinador siguiente hermano + div + p Selecciona el primer elemento <p> que se encuentra directamente por debajo del elemento <div>
Selector lista , div, p Selecciona TODOS los elementos <p> y TODOS los elementos <div>
