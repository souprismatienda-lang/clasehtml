Etiqueta de lista

Las etiquetas de listas nos permiten crear listas ordenadas o listas desordenadas

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

La etiqueta ul, es para crear listas sin orden, cada item de la lista debe estar entre las etiquetas "li" (list item)

  1. Seccion x
  2. Seccion x
  3. Seccion x

Las listas ul y ol soportan atributos para cambiar sus estilos o estados.

Por ejemplo si establecemos el atributo type, podemos cambiar los numeros por letras

  1. Item
  2. Item
  3. Item

De la misma manera, podemos cambiar tambien el estilo de la etiqueta ul con el atributo style

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

Formatos simples

Los formatos simples son etiquetas utilizadas para dar subrayados, negritas, entre otros.

Estos suelen utilizarse dentro de etiquetas que contienen textos. La etiqueta b pone el texto en negrita La etiqueta u pone el texto subrayado

Hipervinculos

Los hipervinculos nos permiten enlazar diferentes documentos o sitios web, tambien se pueden utilizar para referenciar diferentes secciones del mismo documento para crearlos se utiliza la etiqueta a (anchor) junto con el atributo href
