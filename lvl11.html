Variables

Una variable, funciona como un contenedor donde almacenamos valores, estos valores son de diversos tipos, pueden ser numericos, alfanumericos, logicos, entre otros.
En JS hay cuatro maneras de definir las variables.

  • Utilizando let
  • Utilizando const
  • Utilizando var (No es recomendado, se utilizaba en versiones anteriores de JS)
  • Automaticamente (PARA NADA recomendado, siempre declarar variables con su correspondiente sentencia)

Identificadores

Los identificadores son los nombres que les damos a las variables.
Las variables tienen que ser identificadas de manera unica. No pueden existir 2 variables con el mismo identificador.

Los nombres pueden ser cortos como x, y, z. Pueden ser descriptivos como edad, suma, nombreAlumno.

Las reglas para construir nombres (identificadores) son:

  • Pueden contener letras, digitos guiones bajos y signos de peso
  • Deben comenzar con una letra, un $ o un _
  • Son case sensitive (X no es igual a x)
  • No utilizar palabras reservadas (let, const, if, etc.)

¿Cuando utilizar var, let o const?

  1. SIEMPRE declarar variables.
  2. Utilizar const cuando el valor NO va a ser cambiado.
  3. Utilizar const SI el tipo no será cambiado (arrays y objetos).
  4. Utilizar let si no se puede utilizar const.
  5. NUNCA utilizar var

Tipos de datos

Las variables de JS pueden tener 8 tipos de datos, pero por ahora, solo pensemos en numeros y cadenas de caracteres

Las cadenas son texto escrito entre comillas (dobles o simples), y los numeros se escriben sin comillas. Si ponemos numeros entre comillas, estos seran tratados como una cadena de texto.

Multiples declaraciones en una sola sentencia

Podemos declarar multiples variables en una sola sentencia, comenzando con let o const y separando las variables con una coma

Operadores aritmeticos y de asignación

En todos los lenguajes de programación, el operador = no significa igual, sino mas bien asignar.

x = x + 5

La sentencia anterior no tiene sentido en matematicas, pero en programación significa que se asigna el valor de x + 5 a x (Se incrementa en 5 el valor de x)

let x = 5 + 2 + 3;
let y = 2 * 10 + 5;
let z = x / y;

Los operadores aritmeticos funcionan de la misma manera que en matematicas.

let x = "Hola" + " " + "Mundo";

Al utilizar cadenas de caracteres, el operador más las concatena

let x = "5" + 2 + 3;
let y = 2 + 3 + "5";

Si escribimos un numero entre comillas, el resto será tratado como una cadena y concatenado

Variables let

En versiones anteriores de JS, las variables no tenian "scope", con la introducción de la palabra reservada let, las varibles obtuvieron un alcance de bloque. Estas deben ser declaradas antes de utilizarse, y no pueden redeclararse en el mismo en el mismo scope.

Varibales const

const fue introducido a JS al mismo tiempo que let, estas variables no pueden ser redeclaradas, no pueden ser reasignadas, y tienen un scope de bloque.

Utilizamos const cuando declaramos un nuevo array, objeto, función, expresión regular ya que sabemos que estas no seran cambiadas.

La palabra const lleva a confusiones, ya que no define un valor constante sino que define una referencia constante a un valor. Por esto no se puede reasignar un valor constante, reasignar un array constante, ni reasignar un objeto constante, pero si se puede cambiar los elementos de un array y cambiar los atributos de un objeto.

Tipos de datos

JS tiene 8 tipos de datos:

Tipo Descripción
String Un texto compuesto por caracteres encerrados entre comillas
Number Un número representando un valor matemático
Bigint Un número entero grande
Boolean Un tipo de dato que representa un verdadero o falso
Object Una colección de datos del tipo llave-valor
Undefined Una variable primitiva sin valor asignado
Null Un valor primitivo representando la ausencia de un objeto
Symbol Un identificador único y primitivo
