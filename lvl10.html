Sintaxis

La sintaxis son las reglas que definen la manera en que se escriben los programas. Al igual que en cualquier idioma hablado, tenemos que mantener una sintaxis correcta para que pueda ser interpretado de manera adecuada.

Valores

La sintaxis de JS define 2 tipos de valores

  • literales (valores fijos)
  • variables (valores variables)

Los valores pueden ser de distintos tipos, numericos, alfanumericos, booleanos, entre otros.
Los literales son valores escritos directamente en el codigo.

Ejemplo de valor numerico literal

1001 (valor entero).
10,5 (valor decimal)

Ejemplo de valor alfanumerico literal

"Hola Mundo!"
'Hola Mundo!'

Solemos llamar a estos valores alfanumericos "cadenas de caracteres"

Las variables por otro lado, son valores que podemos cambiar.
Para ello tenemos que definir la variable con un identificador.

Podemos imaginar las variables como cajas donde alamacenamos valores, antes de utilizarlas tenemos que definirlas con una de estas 2 palabras claves, let y const

let x = 5;
const nombre="Tinelli";

Las palabras claves son "Case sensitive", JS no va a interpretar Let o LET, hay que escribir estas palabras clave de manera correcta.

Variables JS

Como mencionamos anteriormente, las variables son contenedores utilizados para almacenar datos, estas deben ser identificadas con nombres unicos

Definir x como variable
let x;
Asignar el valor 42 a x
x = 42;

Los identificadores tienen algunas reglas:

  • Deben iniciar con una letra, _, o $
  • Pueden contener digitos luego del primer caracter.
  • No puede ser una palabra clave (const, let, if, etc.).
  • Son "Case sensitive", diferencian entre mayusculas y minusculas.

Operadores

Los operadores en JS pueden ser:

de asignacion (=)

let x = 10
let y = 15
let suma = x + y;

(La respuesta se verá en la consola)

Aritmeticos

5 * 10;

Los operadores aritmeticos (+ - * /) son los utilizados para computar valores

(Antes de invocar a una función que utilice getElementById(), debemos asegurarnos de que dicho elemento ya este cargado en el navegador)

Expresiones

Una expresión es una combinación e valores, variables, y operadores, los cuales se computan en un valor.

Esta expresión solo utiliza valores literales.

(10 + 5) * 10;

Las expresiones también pueden utilizar variables.

x * 10;

Las expresiones con cadenas de caracteres, utilizan el operador + para concatenar diferentes cadenas.

"hola" + " " + "mundo";

(Las cadenas de caracteres deben estar encerradas entre comillas dobles o simples dependiendo de la situación.)

Sentencias

Un programa de computadora es una lista de "instrucciones" para ser "ejecutadas" por la computadora. Estas instrucciones en programación las llammamos sentencias.
La mayoria de los programas escritos en JS, contienen muchas sentencias, las cuales son ejecutadas una por una, de manera secuencial en el orden en que fueron escritas.

Las sentencias estan compuestas por: valores, operadores, expresiones, palabras clave, y comentarios.
En esta sentencia, por ejemplo, le decimos al navegador que escriba "Hola mundo" dentro de un elemento HTML con el id="saludo".

document.getElementById("saludo").innerHTML = "Hola Mundo";

Cada sentencia necesita un delimitador que la separe de la siguiente, este es el punto y coma (semicolon ";")

let a, b, c; // Declaramos 3 variables
a = 5; //Asignamos valor 5 a la variable a
b = 6; //Asignamos el valor 6 a la variable b
c = a + b; //Asignamos la suma de a y b a la variable c

Cuando estan separados por ; se permiten multiples sentencias en la misma linea de codigo.

a = 5; b = 6; c = a + b;

Multiples fuentes en linea, suelen omitir el ; ya que el JS no lo requiere, pero su uso es ALTAMENTE RECOMENDADO.

Comentarios

Los comentarios en cualquier lenguaje de programación, tienen especial importancia, ya que, permiten a los desarrolladores expresar sus intenciones o explicar algoritmos directamente sobre el codigo fuente.
Cada lenguaje maneja de manera diferente los comentarios, pero en la mayoria de los casos son similares.
JS tiene 2 maneras de expresar comentarios:

  • Comentario de linea
  • Comentario en bloque

Comentarios de linea

Cualquier texto entre // y el fin de la linea, será ignorado por JS.

// Esta sentencia cambia la cabecera.
document.getElementById("cabecera").innerHTML = "titulo principal";
// Esta sentencia cambia el parrafo
document.getElementById("parrafo").innerText = "Nuevo parrafo";

Comentarios en bloque

Este tipo de comentarios, tambien llamado multi linea, define una porción de codigo para ser ignorada. Cualquier texto escrito entre /* y */ será ignorado por JS

/*
El codigo siguiente, cambiará el elemento con id="cabecera"
Y el parrafo con id="parrafo"
*/
document.getElementById("cabecera").innerHTML = "titulo principal";
document.getElementById("parrafo").innerText = "Nuevo parrafo";
