Etiqueta Estilo

Hasta el momento utilizamos el atributo estilo para modificar las propiedades de los elementos HTML, pero al hacerlo estamos generando codigo "boilerplate".
para evitar este tipo de codigo, en vez de agregar el atributo style a todas las etiquetas podemos crear una etiqueta llamada style y escribir los estilos dentro de esta.

La etiqueta style aplica los fundamentos de css, esto quiere decir que utilizamos una combinaci칩n de propiedades y valores.

Selectores

Existen multiples selectores para modificar el comportamiento de cada elemento HTML.

  • Selector de etiqueta: este selector a TODAS las etiquetas seleccionadas, ej: div{} modifica todos los divs que se encuentran en el documento
  • Selector de clase: Cada etiqueta puede tener el atributo class, este define un nombre de clase para la etiqueta o grupos de etiquetas. Se pueden establecer multiples clases para cada etiqueta.
    En la etiqueta estilo, seleccionamos la clase con un . Ej: para seleccionar la clase contenido escribimos .contenido {}
  • Selector de id: Al igual que el atributo class cada etiqueta puede tener un atributo id, con la diferencia que este atributo identifica de manera unica al elemento HTML.
    En la etiqueta style lo seleccionamos con un #
