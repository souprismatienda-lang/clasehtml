Divisores (div)

El elemento div es un contenedor generico para el contenido. No afecta la visibilidad de sus etiquetas hijas, ni en el diseño (layout) hasta que se le aplica algun tipo de estilo (Style)

Este texto esta contenido dentro de un divisor. Por defecto este no tiene ningun formato

Con el atributo Style pódemos modificar diferentes propiedades de las etiquetas como el color de fondo, color de fuente, alineacion, comportamiento, entre otros. Este atributo tiene la particularidad de que nos permite agregar multiples propiedades, cada una esta compuesta por la propiedad en si, seguido del valor de la propiedad y por ultimo un delimitador (;), para poder agregar mas propiedades.

Zona de práctica

Crear 2 divisores, con 2 frases distintan, cambiar color de fondo, color de texto y alineacion del texto

En el mundo digital actual, el HTML (HyperText Markup Language) se ha convertido en uno de los lenguajes fundamentales para la creación de páginas web. Conocer sus principales características es esencial para desarrolladores y personas interesadas en el diseño y desarrollo web. HTML es un lenguaje de marcado que utiliza etiquetas para estructurar el contenido de una página web, permitiendo la creación de hipervínculos, la inclusión de imágenes, videos y otros elementos multimedia, así como la definición de estilos y formatos. En este artículo, exploraremos las principales características de HTML, desde su sintaxis básica hasta las últimas novedades y tendencias en su uso. Si estás interesado en aprender más sobre HTML y cómo aprovechar al máximo sus funcionalidades, ¡has llegado al lugar adecuado!

HTML, que significa HyperText Markup Language, es el lenguaje de marcado estándar utilizado para crear páginas web. Es esencial comprender los fundamentos básicos de HTML para poder diseñar y desarrollar sitios web de manera efectiva.

El HTML se compone de elementos que se utilizan para estructurar y presentar el contenido de una página web. Estos elementos incluyen encabezados, párrafos, listas, enlaces y muchas otras etiquetas que se utilizan para dar formato y organizar el contenido.

Una de las características más importantes de HTML es su capacidad para enlazar diferentes páginas y recursos. Los enlaces se crean utilizando la etiqueta y pueden ser utilizados para dirigir a los usuarios a otras páginas, archivos de audio o video, direcciones de correo electrónico, entre otros.
