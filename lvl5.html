Diseño de Grillas

El modulo de diseño de grilla ofrece un diseño basado en grillas con filas y columnas. Este modulo permite a los desarroladores crear facilmente diseños web complejos. Tambien facilita la estructura responsiva sin utilizar float o posicionamiento. Las propiedades CSS de grilla son soportadas por todos los navegadores modernos.

Grilla vs. Flexbox

El diseño de grilla debe ser utilizado para diseños bidimensionales, con filas y columnas.
El diseño Flexbox debe ser utilizado solamente para diseños unidimensionales, con filas o columnas.

Componentes de Grilla CSS

Una grilla siempre consiste de:

  • Contenedor de grilla: este es el elemento <div> padre
  • Item de grilla: son los elementos dentro del contenedor de grilla.

Siempre comenzamos con el elemento padre llamado contenedor de grilla este puede contener uno o varios elementos de grilla. Todos los hijos directos del contenedor de grilla se convierten automaticamente en elementos de grilla.

Filas y columnas

Las lineas verticales son denominadas columnas.

Las lineas orizontales son filas.

El espacio entre las filas/columnas las llamamos gap (salto).
Podemos ajustar el tamaño del gap con las siguientes propiedades de CSS.

  • Column-gap: Espacio entre columnas.
  • row-gap: espacio entre filas.
  • gap: espacio entre fila/columna.

Lineas de Grilla

Las lineas entre columnas son llamadas lineas de columna.
Las lineas entre filas son llamadas lineas de fila.

Podemos especificar donde comienza y termina un item de grilla utilizando estas propiedades:

  • grid-column-start
  • grid-column-end
  • grid-row-start
  • grid-row-end
  • grid-column
  • grid-row

Pistas de Grilla (Grid Tracks)

Las filas y columnas de una grilla son definidas con las propiedades grid-template-rows y grid-template-columns (se pueden resumir como grid o grid-template).
Estas definen las pistas de la grilla (grid tracks). Una pista es el espacio entre 2 lineas de grilla adyacentes.

La propiedad grid-template-columns

La propiedad grid-template-columns establece el número de columnas en el diseño de grillas, y puede definir el ancho de cada columna.
El valor de esta propiedad es una lista separada por espacios, donde cada valor individual definen el ancho de la respectiva columna.
Si queremos un diseño de grilla con 4 columnas, especificamos el ancho de cada columna como auto para que todas las columnas tengan el mismo ancho

Unidad de medida de celdas (fr)

La unidad fr se puede utilizar para definir el ancho de las grillas, de la misma manera que cualquier otra unidad de medida como %, px o em.
La unidad fr significa "fracción". Esta unidad divide automaticamente el espacio disponible en fracciones.
Ejemplo: 1fr tomara una frección del espacio disponible, mientras que 2 fr tomara 2 fracciones del espacio disponible.

Aquí cada columna tomara un 25% del ancho del contenedor, dividido en iguales
