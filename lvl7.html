¿Que es el diseño responsivo?

El diseño responsivo no es un programa ni JavaScript, es una manera de utilizar solo HTML y CSS para poder representar de manera correcta nuestro sitio en todos los dispositivos.
Las paginas web pueden ser visualizadas en diferentes dispositivos como: laptops, pc, tablets o smarthphones. Nuestros sitios deberian verse bien y ser de facil acceso independientemente de su resolución.
Las paginas no tienen que dejar afuera información si no entran en la pantalla, deben adaptarse para encajar en cada una de ellas.

Viewport

El viewport es el area visible de nuestra web para el usuario. este varia con el dispositivo, es mas pequeño en smartphone que en una pantalla de computadora.
Antes de las tablets y los dispositivos mobiles, las paginas web eran diseñadas solo para monitores de PC, y era común tener diseños estaticos y de tamaño fijo. Entonces comenzamos a surfear por internet en diferentes aparatos, las paginas fijas eran demasiado grandes para encajar en el viewport. Para solucionar esto, los navegadores en esos dispositivos escalaban el sitio entero para adecuarlos a su pantalla.

Configurando el Viewport

Este metodo fue introducido en HTML5 para permitir a los diseñadores tomar control sobre la pantalla del usuario, a travez de la etiqueta <meta>. Esta etiqueta debe estar contenida dentro del elemento <head>

El elemento <meta> configurado con el viewport le da instrucciones al navegador para controlar el escalado y las dimensiones de la pagina.

  • width=device-width: establece el ancho de la pagina para que siga el ancho del dispositivo.
  • initial-scale=1.0: establece el zoom inicial cuando la pagina es cargada por primera vez por el navegador.

Escalar contenido al Viewport

Los usuarios estan acostumbrados a navegar los sitios de manera vertical, en mobil o escritorio, pero no horizontalmente.
Entonces, si el usuario es forzado a navegar horizontalmente, o a hacer zoom, para ver el sitio entero, esto resulta en una pobre experiencia del usuario.
Algunas reglas a seguir son:

  1. No utilizar elementos grandes con anchos fijos - Por ejemplo, si el ancho de una imagen es mas grande que el Viewport, esto causa que la pagina se desplace horizontalmente. Recuerda ajustar este contenido para encajar con el ancho del Viewport
  2. No dejar que el contenido dependa de un ancho en particular para renderizarse correctamente - Ya que las dimensiones de pantalla y su ancho en px puede variar mucho entre dispositivos, el contenido no debe depender de un ancho de Viewport en particular. A veces es mejor utilizar medidas relativas y no absolutas.
  3. Utilizar las consultas de medios de CSS para aplicar diferentes estilos para pequeñas o grandes pantallas. - Establecer anchos grandes y absolutos para los elementos de la pagina causara que dichos elementos sean muy grandes para dispositivos pequeños. En vez de esto, considere utilizar valores relativos como width:100%. También hay que ser cuidadosos utilizando valores absolutos para el posicionamiento, ya que podría causar que los elementos caigan fuera del Viewport en dispositivos pequeños.

Media Queries

Media query es una tecnica introducida en CSS3. Utiliza la regla @media para incluir un bloque de propiedades CSS, solo si se cumplen ciertas condiciones.

Agregar un breakpoint

Anteriormente creamos una pagina con filas y columnas, y era responsiva, pero no se veia bien en pantallas pequeñas.
Los media queries nos ayudan con esto. Podemos agregar breakpoints (puntos de control), donde ciertas partes del diseño se comportaran de manera diferente en cada punto.

Puntos de quiebre tipicos para diferentes pantallas

Hay muchos dispositivos con diferentes alturas y anchos, por ello es dificil crear un punto exacto por cada dispositivo. Para mantener las cosas sencillas podemos enfocarnos en 5 grupos:

  • @media only screen and (max-width: 600px){....} : Dispositivos extra small (telefonos, 600px para abajo)
  • @media only screen and (min-width: 600px){....} : Dispositivos small (tablets y telefonos grandes, 600px para arriba)
  • @media only screen and (min-width: 768px){....} : Disposivos medium (tablets panoramicas, 768px para arriba)
  • @media only screen and (min-width: 992px){....} : Disposivos large (laptops/PCs, 992px para arriba)
  • @media only screen and (min-width: 1200px){....} : Disposivos extra large (laptops grandes y PCs alta resolución: 1200px para arriba)
