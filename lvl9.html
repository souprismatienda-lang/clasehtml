¿Que es JavaScript?

JavaScript es el lenguaje por defecto de programación en la web, puede calcular, manipular y validar datos. Se puede utilizar para modificar o actualizar ambos codigos HTML y CSS.
Si bien comparte nombre con el lenguaje Java, estos no son iguales. JavaScript se envia desde el servidor web a travez de los mensajes http junto con el html, y este se ejecuta en el navegador cliente, a diferencia e Java que se compila en la PC.

JavaScript puede cambiar el contenido HTML

Uno de los muchos metodos de JavaScript con HTML es getElementById(). Este metodo "encuentra" un elemento HTML cuyo ID sea igual al que definimos entre parentesis (innerHTML).

¿Donde colocamos JS?

La etiqueta <script>

Esta etiqueta funciona de manera similar a <style>, nos permite agregar codigo js fuera de los atributos de los elementos HTML.

Funciones y Eventos

Una función de JavaScript es un bloque de codigo que puede ser ejecutado cuando es "invocado".
Por ejemplo: Una función puede ser invocada cuando ocurre un evento, como cuando el usuario hace click en un boton.

Scrips en <head> o <body>

Podemos colocar cualquier cantidad de scripts en un documento HTML, estos pueden ir dentro de la sección body o la sección head.

Salidas JavaScript

Existen muchas maneras de mostrar información en pantalla a travez de JavaScript. Ya vimos que podemos acceder a un elemento HTML con el metodo document.getElementById(id), este selecciona el elemento cuyo id coincida con el argumento enviado (los argumentos son los valores que ponemos entre parentesis de la función). Una vez seleccionado, modificamos HTML con el metodo innerHTML o el texto con innerText, pero hay otras maneras de expresar información al usuario.

document.write()

Este metodo sobreescribe el documento entero, por lo tanto nunca hay que invocarlo luego de que se haya cargado la pagina.

El principal uso de este metodo es el de probar diferentes cosas.

window.alert()

Esta es una ventana de "alerta", cada vez que se invoca, muestra un "popup" con el mensaje contenido en el argumento de la función.

El estilo de esta ventana, va a depender del sistema operativo y del navegador en el que se haya cargado.
Adicionalmente la palabra window puede ser omitido.

En JavaScript, el objeto window tiene un alcance global, esto quiere decir que las variables, propiedades y metodos, por defecto pertenecen al objeto window. Por ello, especificar window es opcional.

console.log()

Utilizado para propositos "debugging", este muestra en la consola el mensaje.

window.print

Sirve para imprimir la pagina completa
